###########하드링크와 소프트링크##########
	
ln 타켓파일 하드링크이름
- 링크파일 만들기(기본설정: 하드링크)

'stat 파일이름' 
- inode 란에서 하드링크와 원본 파일값이 똑같다는 것을 확인 가능
- linked 란에서 링크된 갯수 확인 가능

hard link: hello_world 내용을 바꾸면 hello_hardlink의 내용도 바뀌어있다

ln -s
- 소프트링크 만들기
- inode값이 달라졌다
- 파일의 크기도 다르다
- -> 소프트링크 파일에는 원본 파일의 '경로'가 저장되어 있다 
- 내용을 바꾸면 원본 파일 내용도 바뀜
